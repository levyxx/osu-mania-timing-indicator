# Icon Placeholder

アイコンファイルをこのディレクトリに配置してください：

- **Windows**: `icon.ico` (256x256推奨)
- **Linux**: `icon.png` (512x512推奨)
- **macOS**: `icon.icns`

アイコンがない場合でも、Electronのデフォルトアイコンでビルドできます。

## アイコン作成ツール

- [icoconverter.com](https://icoconverter.com/) - PNG to ICO/ICNS変換
- [cloudconvert.com](https://cloudconvert.com/) - 各種フォーマット変換
